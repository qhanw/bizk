## 安装

此处示例采用`pnpm`安装，也可用`yarn`、`npm` 替换`pnpm`。

### @bizk/store

```bash
pnpm add @bizk/store
```

### @bizk/utils

```bash
pnpm add @bizk/utils
```

### @bizk/regexps

```bash
pnpm add @bizk/regexps
```

## 用法

详细用法请查看文档：https://bizk.qhan.wang/

## Development

```bash
# install dependencies
$ pnpm i

# start dev server
$ pnpm start

# build docs
$ pnpm run build
```
