# 工具函数

业务开发中常用的，基本公共函数处理方法，用以提升业务开发效率，统一函数使用规范。

### 安装方式

推荐使用`pnpm`作为包管理工具。当然也可使用`npm`或者`yarn`。

```bash
pnpm add @bizk/utils
```

### 功能函数

- calcTableOptWidth
- renameKeys
- arrayToTree (coming soon)
- treeToArray (coming soon)
