# Hooks

基于`antd`额外扩展配置的常用业务组件，用以提升业务开发效率，避免同一类似功能组件多次实现。

### 安装方式

推荐使用`pnpm`作为包管理工具。当然也可使用`npm`或者`yarn`。

```bash
pnpm add @bizk/hooks
```

### Hooks 列表

- useServerTableSelect 服务端全选
- useTheadStretch 表格表头伸缩
- useQueryParams 查询参数解析 (coming soon)
